Suporte – Popcorn Movies

1. Perguntas Frequentes (FAQ)

Como encontrar um filme ou série?

Use a barra de busca na página inicial para digitar o nome do título desejado.

De onde vêm as informações dos filmes?

As informações são obtidas via integração com a API pública do TMDB.

Preciso me cadastrar para usar o site?

Não. O Popcorn Movies é um projeto educacional e não exige cadastro.

2. Problemas Técnicos

Se uma imagem não carregar, tente atualizar a página ou verificar sua conexão com a internet.

Se um botão ou link não funcionar, entre em contato conosco pela página de contato
